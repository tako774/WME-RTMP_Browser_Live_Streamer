更新履歴：WME Browser Live Streamer (WMEブラウザ配信ツール)

○Ver.0.14
- Firefox 21 以降で、Windows Media Player プラグインを有効化する手順の説明を追加

○Ver.0.13
- 2013/4/5 以降、twitter 認証が正常にできなくなっていたのを修正
  ACCESS TOKEN 画面を開く際の js で、PIN を取得しそこなっていた不具合を修正しました。
  これが原因で、ACEESS TOKEN 取得時に oauth_verifier が空文字列になっていました。
  2013/4/5 頃の twitter 仕様変更前までは、oauth_verifier は必須でなかったのが、
  同日以降必須となったため、正常に ACCESS TOKEN が取得できなくなっていました。

○Ver.0.12
- index.html の title を空から「WMEブラウザ配信ツール」に変更
　検索サイトは、javascript で挿入しているページタイトルを認識しないので、
　検索にひっかかるようにしたい場合には、index.html のタイトルを自力で修正願います。
　（いまのところ更新のたびに index.html を上書きしたあと、自力で修正する形になります）
- img タグのとじ忘れ修正
- ツイート記入欄の残り文字数表示について、URLの場合の計算文字数を22(http),23(https)に修正

○Ver.0.11
- 認証直後だと、コメント欄の自動更新が動作しないのを修正
- クッキー（Twitterの認証保存用）の有効期限を1年から20年に変更
- 認証済の場合には再認証の表示がでないように修正
- ツイート表示欄で、名前が長いと時刻表示部分が下にずれてしまうのを修正
- 24時間以上前のtwitterのツイート時間文字列の生成にバグがあったので修正
- Opera だと Twitter の PIN 入力で 0 から始まる値が入力できないのを修正

○Ver.0.10
- Twitter API 1.1 に対応
  - コメント欄（ハッシュタグ検索結果）を含め、すべてユーザーの OAuth 認証が必須に
  　cookie で認証情報は保存されるので、一回認証すれば cookie が有効な限り再認証は不要です
  - ツイート、RT、fav の際に twitter の画面に飛ばされなくなった

○Ver.0.06
- Ver.0.05 での上２つの更新が入っていなかったので修正
- 鏡モード時、配信親からのツイートは「親」と表示され、配信親にリンクが貼られるようにした

○Ver.0.05
- 鏡モードの時に、ツイートの放送URLが表示されていたのを、略すように修正
- 鏡からのツイートの時に、「鏡」と表示され、鏡へのリンクが貼られるようにした
- ハッシュタグがマルチバイト文字の際、URIエンコードしてURLに渡すようにした
　（IE等で検索の結果がおかしくなる不具合があったのが直りました）

○Ver.0.04
- アップデートがあった場合の通知を追加（わりとかなり控えめに、最新Ver.を表示します）
- HTML上の配信ツール自体に関する記述の詳細化
- 配信ツール自体に関する記述の位置をフッターからツイートリストの下に移動
- ツイート時間部分の閉じタグ不整合を修正
- IE以外でプラグイン未インストール時の文章をより丁寧に

○Ver.0.03
- 鏡モード追加
	URLの後ろに "?<他のWMEブラウザ配信ツールのURL>"と入れると、
	他の配信の config.js を拾ってきて、適用してくれます
	ただし、配信元URL(WMEないし鏡のURL)だけは、自分の config.js のものをつかいます
- 8時間以上前のツイートをグレー表示
- 8時間以上間の空いたツイート間を二重線に
- ツイートの時間部分に twitter の status へのリンクを追加
- 配信ページHTMLのデフォルトファイル名を bc.html -> index.html に変更
- 配信ページに開発レポジトリ／更新履歴のリンク追加

○Ver.0.02
- 配信サイズの可変化; config.js で配信サイズが設定できます
- 配信元URL（外部プレイヤー・鏡用）を右下に表示するように
- 配信内容説明の設定を config.js から description.txt に分離 
- 配信内容説明を、ブラウザからリロードなしで自動更新されるように

○Ver.0.01
- 初リリース
- WMEのストリーミングをブラウザで見る機能
- twitter の特定のハッシュを画面に準リアルタイムで表示する機能
